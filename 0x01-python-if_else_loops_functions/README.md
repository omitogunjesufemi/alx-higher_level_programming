# 0x01. Python - if/else, loops, functions

This project is to know how important indentation is in Python, how to use the `if`, `if...else` statements, how to use comments, how to affect values to variables, how to use the `while` and `for` loops, how Python's `for` is different from C's, how to use the `break` and `continues` statements, how to use the `else` clauses on loops, what the `pass` statement does and when to use it. It also teaches how to use `range`, what a function is and how to be used, what is return by a function when it does not use any return statement, scope of variables, traceback, what the arithmetic operators are and how to use them.
